Регистрация по почте

Для прохождения регистрации по почте, следуйте указанным шагам:

Перейдите по пути /signup/ для регистрации.
После регистрации, подтвердите свою почту по пути /verify-email/. (вам на email придет код или же в админке посмотрите)
Установите пароль по пути /set-password/.
Также в проекте реализован сброс пароля и возможность отправить пароль повторно.

Вход и аутентификация

Все запросы в проекте осуществляются посредством токена. Если вы используете Postman, сначала выполните запрос /signin/ для получения токена. При выполнении запросов укажите полученный токен в заголовке Authorization как Bearer токен.

Этот подход обеспечивает безопасность и предотвращает выполнение запросов неавторизованными пользователями.
